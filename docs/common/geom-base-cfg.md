options:

| 参数名   | 类型    | 默认值 | 描述               |
| -------- | -------  | ------ | ------------------ |
| sortable | _boolean_  | -      | 是否对数据进行排序 |
| theme    | _object_   | -      | 主题配置           |
| visible  | _boolean_    | -      | 是否可见           |
| useDeferredLabel ⭐️ | _boolean\|number_    | false     |延迟渲染 Geometry 数据标签. 设置为 true 时，会在浏览器空闲时期被调用，且默认 timeout 为 100ms, 也可以指定具体 timeout 时间          |
