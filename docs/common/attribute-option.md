_AttributeOption_ 配置如下：

| 参数名   | 类型             | 是否必选 | 默认值 | 描述                                        |
| -------- | ---------------- | -------- | ------ | ------------------------------------------- |
| fields   | string[]         |          | -      | 映射的属性字段                              |
| callback | (...args) => any |          | -      | 回调函数，参数为配置的字段，return 配置内容 |
| values   | any[]            |          | -      | 指定常量映射规则                            |
